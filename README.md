# Pizza Sales Performance Dashboard | Power BI

## Project Overview
This project showcases an interactive Power BI dashboard built to analyze pizza sales performance and uncover key business insights. The dashboard helps stakeholders track revenue, order trends, customer behavior, and product performance.

## Key Metrics
- Total Orders: 21K
- Average Pizzas per Order: 2.32
- Total Revenue: $817.86K

## Visualizations
- Revenue trend over time
- Revenue by pizza category
- Quantity sold by pizza size
- Top-selling pizzas by revenue

## Insights
1. Large pizzas are most popular.
2. Classic pizza category generates slightly higher revenue.
3. “The Thai Chicken Pizza” is the top revenue contributor.
4. Revenue shows a dip in the last quarter (Oct 2015).
